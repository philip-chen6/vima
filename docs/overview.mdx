Vima turns egocentric construction footage into auditable spatial evidence.
Instead of asking a model to guess from raw video, Vima builds a compact evidence
layer first, then exposes it to humans and agents through a dashboard, HTTP API,
CLI, and MCP server.

<CardGroup cols={2}>
  <Card title="Evidence First" icon="scan-search">

    context before they become an answer.
  </Card>
  <Card title="Agent Native" icon="bot">

  </Card>
  <Card title="Small Production Footprint" icon="server">

  </Card>

## Pipeline

```text
Hardhat frames
  -> precomputed spatial inference
  -> CII work labels
  -> COLMAP zone attribution
  -> cited dashboard / API / agent response
```

## Current Hosted Facts

- 30 sampled masonry frames
- 86.7% wrench time
- Cached CII frame rows and spatial zones
- Offline inference artifacts served through a small hosted API

<Note>
